# Dicoding Collection Dashbord

## Installation
Open a terminal and run::

```bash 
$ pip install pandas
$ pip install numpy
$ pip install matplotlib
$ pip install seaborn
$ pip install sklearn
$ pip install streamlit
$ streamlit hello
```

## Quickstart
1. Create a new file app.py with the following code:
```bash 
import streamlit as st

print("Hello World")
```

2. Run streamlit app
```bash 
streamlit run app.py
```
